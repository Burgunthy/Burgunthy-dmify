# CommentLink

인스타그램 댓글 → 자동 팔로우 확인 → 제휴링크 DM 발송 SaaS

## Stack

- **Frontend**: Next.js 15 + Tailwind CSS
- **Backend**: Next.js API Routes (Serverless)
- **Database**: Supabase (PostgreSQL)
- **Hosting**: Vercel
- **Webhook**: Instagram Graph API

## Dev

```bash
npm install
npm run dev
```

## Deploy

Push to `master` → auto-deploy via Vercel
